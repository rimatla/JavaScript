JS is a 'single threaded' language - Its engine can only execute one piece of code/task at a time.
ex: A Bank w/ one teller or a Bar with only one bathroom.

JS Event Loop - Contains a queue for the subsequent code to be executed.
ex: a line waiting  for the next available bank teller

JS Non-Blocking - Allows for the next code/task to be executed while the previous one may be still processing.
